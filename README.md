# MyLocalGarage

A full-stack PHP web application for discovering and managing local garages.

## Features
- Public garage search
- Booking form with database storage
- Admin authentication & job management
- Secure password hashing
- MVC-style project structure
- Local development environment with PHP built-in server

## Tech Stack
- PHP 8.x
- MySQL (PDO)
- HTML / CSS
- Nginx (production)
- Git & GitHub (workflow)

## Project Status
Actively developed as a portfolio project.
